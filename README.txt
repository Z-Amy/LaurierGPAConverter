Description: An Excel macro that takes a Laurier formatted transcript in PDF format.
It converts the grades from a 12.0 scale to a 4.0 scale and calculates the converted GPA

How to use: Download the .xlsm file I have included and press the "Upload Transcript" button to run the macro.
I have also exported the macro as a .bas file and included it in the repository. 

Tools: Excel VBA
